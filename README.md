# Code.HP - Retro Code Snippet Manager

A retro-futuristic alien-themed code snippet manager built with Electron.

## 🚀 launch app
Run Code.HP Setup 1._._.exe
Double-click the desktop shortcut "Code.HP" to open the app.

## ✨ Features
- Retro-futuristic alien UI with VT323 font
- Monaco editor with VS Code syntax highlighting
- Support for 25+ programming languages
- File-based snippet storage
- Real-time search functionality
- Dark theme with green accents

## 📁 File Structure
- `snippets/` - Your code snippets are stored here
- `Code.HP.vbs` - Silent launcher script
- `create-desktop-shortcut.ps1` - Creates desktop shortcut

## 🛠️ Manual Launch
If needed, run: `npm start`
